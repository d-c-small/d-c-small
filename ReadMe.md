# About Me:
As a Data Analyst II at Gray Inc., I design and deploy custom LLM-powered agents using GitHub Copilot and Model Context Protocol (MCP) — tools that automate database design workflows, enforce SQL coding standards, and give data teams their time back. I've also built out ~50 database objects across a medallion architecture spanning 4 source systems on Azure Managed Instance, so I'm just as comfortable using SQL as I am designing AI-powered workflows.<br><br>My background is anything but traditional. I've worked in environmental data management, law enforcement, and now AI-focused data engineering. Every pivot taught me something: how to break down complex problems, how to communicate with non-technical stakeholders, and how to build systems that people actually use.<br><br>A few things I'm especially passionate about:<br>→ Making AI accessible to teams through hands-on workshops and documentation<br>→ Designing data architectures that are both human-readable and AI-ready<br>→ Building governance frameworks so AI adoption is sustainable, not chaotic<br>→ Turning data into visual stories with Power BI and Python<br><br>I'm always up for a conversation about AI agents, data engineering, or unconventional career paths into tech.
